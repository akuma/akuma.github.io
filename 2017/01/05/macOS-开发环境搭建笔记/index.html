安装其他命令行工具

一些工具说明：

  • coreutils：GNU core utilities
  • moreutils：GNU more utilities
  • autojump：目录导航命令
  • htop：类似 top 的命令
  • git-extras：git 扩展命令
  • diff-so-fancy：更好看的 diff 工具
  • mycli：MySQL 命令行客户端
  • cloc：统计代码行数的工具
  • tmux：多终端模拟工具
  • tree：显示目录树的命令
  • unrar：解压 rar 文件的命令
